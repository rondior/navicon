# Navicon

Navicon is a customizable visual New Tab dashboard for Chrome.

It replaces your default new tab page with a fast, elegant, distraction-free productivity workspace.

---

## ✨ Features

### Core (Free)
- Clean visual tile grid
- Drag-and-drop reordering
- Adjustable tile sizing
- Theme system
- Local-first storage (no tracking, no accounts)

### Advanced Layouts
- Sections layout (organized dashboard view)
- Folders layout (grouped tile containers)

## 🛡 Data & Privacy

Navicon does **not** collect, transmit, sell, or share user data.

All tiles and settings are stored locally using `chrome.storage.local`.

No analytics.
No background tracking.
No external servers.

---

## 🔑 Permissions

- `storage` — required to save tiles and user settings locally.

## 🛠 Development

Navicon is built with:

- Vanilla JavaScript
- CSS
- HTML
- Chrome Extension Manifest V3

To run locally:

1. Clone the repository
2. Navigate to `chrome://extensions`
3. Enable Developer Mode
4. Click "Load Unpacked"
5. Select the project folder
